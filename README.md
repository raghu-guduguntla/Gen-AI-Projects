# Gen AI Projects

Exploring Gen AI through small projects focused on solving interesting daily problems.

## About
This is a personal repository where I experiment with AI agent ideas, patterns, and workflows.  
Each project is small by design and focused on learning through practical problem-solving rather than building production-ready systems.

## Goals

- Explore different architectures and interaction patterns
- Understand trade-offs between autonomy, control, and reliability
- Keep things simple, practical, and fun

## Structure
Each folder typically represents a standalone experiment or idea, with:
- A short problem description
- The approach taken
- Implementation code
- Observations and learnings
